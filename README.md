# Elevate Labs Task 3 - SQL for Data Analysis

## Overview
This project was completed as part of the **Elevate Labs SQL Internship – Task 3**. It demonstrates the use of SQL for database creation, data manipulation, querying, joins, views, indexes, aggregate functions, and subqueries using **MySQL Workbench**.

---

## Tools Used
- MySQL Workbench 8.0
- MySQL Database

---

## Database Schema
The project consists of the following tables:

- Categories
- Products
- Customers
- Orders
- Order_Items

These tables are connected using **Primary Keys** and **Foreign Keys** to simulate an E-Commerce database.
